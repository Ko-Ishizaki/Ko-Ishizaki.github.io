Dummy Image Generator Custom placeholder image service

https://dummyimage.kdf-jp.tokyo
Technologies: Node.js, Express.js
This is a self-hosted dummy image generator built as an alternative to via.placeholder.com, which stopped functioning properly. It provides dynamically generated placeholder images with customizable text, dimensions, and colors.
The base domain dummyimage.kdf-jp.tokyo redirects to ko-ishizaki.github.io when accessed directly, but image URLs work as expected when structured correctly.

Example usage:
https://dummyimage.kdf-jp.tokyo/400x200?text=Test1&bg=white&fg=black&fontSize=100

URL structure:

  • /{width}x{height} image dimensions and format (currently PNG supported)
  • ?text=YourText text to be displayed (URL-encoded)
  • &bg=BackgroundColor background color (HTML color name or hex code)
  • &fg=TextColor text color (HTML color name or hex code)
  • &fontSize=Size font size in pixels

Color values can be standard CSS color names like white, black, or hexadecimal codes such as #ffcc00. The system is designed to be simple and flexible for developers needing placeholder images in prototyping or documentation.
